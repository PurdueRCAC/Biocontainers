Mcl
==============================

Introduction
~~~~~~~~
Mcl is short for the Markov Cluster Algorithm, a fast and scalable unsupervised cluster algorithm for graphs.


| For more information, please check:
| BioContainers: https://biocontainers.pro/tools/mcl 
| Home page: http://micans.org/mcl/

.. note::
    Please follow the recommended citation guidelines from the developers when you use the tool in research.

Module
~~~~~~~~
You can load the modules by::

    module load biocontainers
    module load mcl

Example job
~~~~~
.. warning::
    Using ``#!/bin/sh -l`` as shebang in the slurm job script will cause the failure of some biocontainer modules. Please use ``#!/bin/bash`` instead.

To run mcl on our clusters:

.. tab-set::

  .. tab-item:: Anvil

    .. code-block:: bash

        #!/bin/bash
        #SBATCH -A myallocation     # Allocation name
        #SBATCH -p wholenode        # Partition name
        #SBATCH -t 1:00:00
        #SBATCH -N 1
        #SBATCH -n 1
        #SBATCH --job-name=mcl
        #SBATCH --mail-type=FAIL,BEGIN,END
        #SBATCH --error=%x-%J-%u.err
        #SBATCH --output=%x-%J-%u.out

        module --force purge
        module biocontainers mcl

        # Your mcl workflow...

  .. tab-item:: Bell, Gautschi, or Negishi

    .. code-block:: bash
