Moshpit
==============================

Introduction
~~~~~~~~
MOSHPIT (MOdular SHotgun metagenome Pipelines with Integrated provenance Tracking) is the QIIME 2 distribution for whole-metagenome shotgun (WMS) analysis. It provides q2-assembly and q2-annotate plus accessory plugins for read QC, contig assembly (MEGAHIT/SPAdes), assembly evaluation (metaQUAST, BUSCO), binning, taxonomic classification (Kraken 2, Kaiju, mOTUs 3), and functional annotation (eggNOG + DIAMOND), all with full QIIME 2 provenance tracking. The single command-line entry point is "qiime".

Module
~~~~~~~~
You can load the modules by::

    module load biocontainers
    module load moshpit

Example job
~~~~~
.. warning::
    Using ``#!/bin/sh -l`` as shebang in the slurm job script will cause the failure of some biocontainer modules. Please use ``#!/bin/bash`` instead.

To run moshpit on our clusters:

.. tab-set::

  .. tab-item:: Anvil

    .. code-block:: bash

        #!/bin/bash
        #SBATCH -A myallocation     # Allocation name
        #SBATCH -p wholenode        # Partition name
        #SBATCH -t 1:00:00
        #SBATCH -N 1
        #SBATCH -n 1
        #SBATCH --job-name=moshpit
        #SBATCH --mail-type=FAIL,BEGIN,END
        #SBATCH --error=%x-%J-%u.err
        #SBATCH --output=%x-%J-%u.out

        module --force purge
        module biocontainers moshpit

        # Your moshpit workflow...
