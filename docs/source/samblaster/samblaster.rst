Samblaster
==============================

Introduction
~~~~~~~~
Samblaster is a tool to mark duplicates and extract discordant and split reads from sam files.


| For more information, please check:
| BioContainers: https://biocontainers.pro/tools/samblaster 
| Home page: https://github.com/GregoryFaust/samblaster

.. note::
    Please follow the recommended citation guidelines from the developers when you use the tool in research.

Module
~~~~~~~~
You can load the modules by::

    module load biocontainers
    module load samblaster

Example job
~~~~~
.. warning::
    Using ``#!/bin/sh -l`` as shebang in the slurm job script will cause the failure of some biocontainer modules. Please use ``#!/bin/bash`` instead.

To run samblaster on our clusters:

.. tab-set::

  .. tab-item:: Anvil

    .. code-block:: bash

        #!/bin/bash
        #SBATCH -A myallocation     # Allocation name
        #SBATCH -p wholenode        # Partition name
        #SBATCH -t 1:00:00
        #SBATCH -N 1
        #SBATCH -n 1
        #SBATCH --job-name=samblaster
        #SBATCH --mail-type=FAIL,BEGIN,END
        #SBATCH --error=%x-%J-%u.err
        #SBATCH --output=%x-%J-%u.out

        module --force purge
        module biocontainers samblaster

        # Your samblaster workflow...
